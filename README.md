# Avalanche Project

Este projeto foi inicializado com suporte para:
- **GitHub**: Integração via Decokit para manipulação de repositórios.
- **Firebase**: Configurado para Auth e Firestore.
- **Google Stitch**: Integrado via StitchMCP (Projeto ID: 6250884992993881202).

## Estrutura de Conectores
Os conectores estão localizados em `src/services/`:
- `github.js`
- `firebase.js`
- `stitch.js`

## Configuração
Renomeie o arquivo `.env.example` para `.env` e preencha as credenciais necessárias.

## Instalação
Para instalar as dependências, execute:
```bash
npm install
```
